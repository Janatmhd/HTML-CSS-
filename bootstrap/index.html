Bootstrap

Introduction

Bootstrap est un framework CSS. Un framework est un ensemble de librairies regroupées dans un but précis et possédant des règles internes que doivent suivre les utilisateurs.
En d'autres termes et pour le dire plus simplement, Bootstrap est un ensemble de fichiers CSS et Javascrip fonctionnant ensemble et qu'on va pouvoir utiliser pour créer des designs complexes de manière relativement simple.
L'interet d'un framework c'est de nous permettre de réaliser plus de choses en codant moins.

Comment utiliser Bootstrap ?

Bootstrap est un ensemble de fichiers CSS et Javascript, donc pour l'utiliser nous devons integrer ses fichiers à notre projet ou tout simplement faire appel à ses fichiers via un CDN (Content delivery network).

  • Première méthode
    • Télécharger Bootstrap depuis le site https://getbootstrap.com/ ensuite integrer les fichiers CSS et Javascript à notre projet comme n'importe quel autre fichier CSS ou Js.
  • Seconde méthode
    • Utiliser un CDN de bootstrap et le lier à notre projet comme n'importe quel autre CDN (réseau de distribution de contenu)

Options de grilles

Bootstrap propose une grille de 12 colonnes. Son système de grille peut s'adapter aux 6 points de rupture (breakpoint) par défaut et à tout autre point de rupture personnalisé par l'utilisateur.
Les 6 niveaux de breakpoint de Bootstrap sont les suivants :

  • Tres petit (xs)
    • écran inferieur à 576px
  • petit (sm)
    • écran supérieur ou égal à 576px
  • Moyen (md)
    • écran supérieur ou égal à 768px
  • Grand(lg)
    • écran supérieur ou égal à 992px
  • très Grand(xl)
    • écran supérieur ou égal à 1200px
  • très très Grand(xxl)
    • écran supérieur ou égal à 1400px
